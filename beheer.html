🔗 Google Calendar koppeling

Als je Google Calendar activeert in script.js, wordt de agenda automatisch bijgewerkt op basis van je Google Agenda-afspraken. De handmatige instellingen hierboven dienen als fallback.

✓ Opgeslagen!
  1. Ga naar console.cloud.google.com
  2. Maak een project aan → Activeer "Google Calendar API"
  3. Maak een API Key aan (beperk tot jouw domein)
  4. Stel je Google Agenda in als Publiek leesbaar
  5. Vul de API Key en Calendar ID hierboven in en klik opslaan
